- Uses kebab-case for documentation files (e.g., `data-fetching.md`, `routing.md`, `ui.md`)
- All documentation stored in `/docs` directory at project root

### CLAUDE.md Structure
- Documentation list located in "## Development Guidelines" section
- List appears after the **IMPORTANT** notice about referring to docs before generating code
- Uses bullet point format: `- /docs/filename.md`
- **Alphabetical ordering** is maintained for easy scanning
- No grouping or categories - simple alphabetical list
